# AI Overlay Assistant

一个始终悬浮在桌面最顶层的轻量 AI 对话窗口，专为需要**在阅读材料的同时快速检索知识点**的场景设计。支持透明度调节，可叠加在任意应用上层使用而不干扰底层窗口。

内置四种面向**计算机专业研究生**的知识检索模式，针对不同类型的知识点提供结构化、可直接参考的输出格式。

---

## 功能特性

- **悬浮置顶** — 始终显示在所有窗口之上，不遮挡主要工作区
- **透明度可调** — 30%～100% 实时调节，与底层内容叠加浏览
- **失焦自动隐藏** — 点击其他窗口后 UI 框架淡出，只保留对话文字悬浮在屏幕上；点击文字区域立即恢复
- **模型切换** — 标题栏一键切换 `Pro`（deepseek-v4-pro）/ `Flash`（deepseek-v4-flash）
- **全局快捷键** — `Ctrl+Space` 随时唤出 / 隐藏，不打断当前操作
- **流式输出** — 打字机效果逐字显示，响应即时
- **Markdown 渲染** — 代码块、列表、标题完整渲染
- **多轮上下文** — 会话内保留对话历史，支持追问
- **清除上下文** — 一键清空对话历史，开始新一轮检索
- **题型 Skill** — 按 `/` 选择检索模式，AI 按对应格式结构化输出

### 四种 Skill 模式

| 模式 | 触发 | 输出格式 |
|------|------|----------|
| 📖 名词解释 | `/` → 选择 | 【定义】+ 【关键点】，≤120字 |
| ✏️ 简答题   | `/` → 选择 | 分点编号，≤150字 |
| 🔧 设计题   | `/` → 选择 | 【思路】+【步骤/代码】+【说明】 |
| 🔢 计算题   | `/` → 选择 | 逐步编号 +【答】最终结果 |

## 环境要求

- Node.js ≥ 18
- Windows 10 / 11
- DeepSeek API Key（在 [platform.deepseek.com](https://platform.deepseek.com) 注册获取，新用户有免费额度）

## 部署步骤

### 1. 克隆或下载代码

```bash
git clone <repo-url>
cd electron-ai-overlay
```

### 2. 安装依赖

```bash
npm install
```

> 首次安装会下载 Electron（约 120 MB），视网络情况需要 1～3 分钟。

### 3. 配置 API Key

```powershell
# Windows PowerShell
Copy-Item .env.example .env
```

用任意文本编辑器打开根目录下的 `.env` 文件，填入 Key：

```
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

> Key 在 [platform.deepseek.com](https://platform.deepseek.com) → 个人中心 → API Keys 页面创建。

### 4. 启动

```bash
npm start
```

窗口以 360×500px 悬浮启动，可拖拽到屏幕任意位置。

## 使用方法

### 窗口操作

| 操作 | 说明 |
|------|------|
| 拖动标题栏 | 移动窗口位置 |
| 拖拽窗口边缘 | 自由调整窗口大小 |
| 透明度滑块 | 实时调节透明度（30%～100%） |
| `Pro` / `Flash` 切换 | 标题栏模型切换按钮，即时生效 |
| `Ctrl+Space` | 全局快捷键，显示 / 隐藏窗口 |
| 点击其他窗口 | UI 自动淡出，只显示对话文字；点击文字区域恢复 |
| `↺`（绿色） | 清除对话历史，开始新会话 |
| `−`（橙色） | 最小化 |
| `×`（红色） | 关闭 |

### 发送消息

- **Enter** — 发送
- **Shift + Enter** — 输入框内换行

### 使用 Skill 模式

1. 确保输入框**为空**，按 `/`
2. `↑` `↓` 选择题型，或直接点击
3. `Enter` 确认，输入框左侧出现题型标签
4. 输入题目内容，`Enter` 发送
5. 点击标签右侧 `×` 清除，恢复自由问答

> Skill 在整个会话中持续生效，随时按 `/` 切换。

### 示例

**名词解释**

```
输入：虚拟内存
```
```
【定义】操作系统提供的内存管理机制，使进程可使用超过物理内存大小的地址空间。
【关键点】
· 通过页表将虚拟地址映射到物理地址或磁盘交换区
· 缺页中断触发页面置换，常用算法有 LRU、Clock
· 实现进程隔离，每个进程拥有独立地址空间
```

**计算题**

```
输入：页框数为 3，LRU 算法，访问序列 1 2 3 4 1 2 5 1 2 3，求缺页次数
```
```
（1）1 → [1] 缺页
（2）2 → [1,2] 缺页
...
【答】共缺页 7 次
```

## 常见问题

**Q：透明度调得太低找不到窗口？**
按 `Ctrl+Space` 隐藏再显示，透明度恢复默认；或重启应用。

**Q：失焦后窗口完全看不到了？**
点击窗口所在屏幕区域（对话文字位置）即可重新聚焦，UI 恢复显示。若确实找不到，按 `Ctrl+Space` 重新呼出。

**Q：提示 `DEEPSEEK_API_KEY not set`？**
检查根目录是否存在 `.env` 文件，Key 须以 `sk-` 开头。

**Q：`Ctrl+Space` 与输入法冲突？**
修改 `main.js` 第 31 行的快捷键，例如改为 `Control+Shift+A`。

**Q：某些全屏应用上无法置顶？**
部分程序（游戏、视频播放器全屏模式）独占显示层，属系统限制。
